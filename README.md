# 辐射超标可视化标注

你现在可以做四件事：
1. **上传报告文件**：`txt / docx / pdf`
2. **自动识别超标点位与手套孔**
3. **初始化障碍物区域**：默认 / 手工输入 ID / 上传掩膜图
4. **在网格图中框选点位并批量定义区域**

## 功能说明

- 自动识别：`辐射超标地面点位ID`
- 自动识别：`辐射超标手套孔名称`
- 按 30 × 28 网格展示，总计 840 个点位
- 支持将选中点位批量设为：
  - 障碍物
  - 待测区域
  - 清除定义

## 安装

```bash
pip install -r requirements.txt
```
## 已解决冲突并合并的功能
1. 上传检测报告（`txt / docx / pdf`）自动识别超标点位与手套孔。
2. 障碍物初始化支持三种方式：
   - 默认障碍物
   - 手工输入障碍物 ID
   - 上传掩膜图（黑=障碍、白=可达）
3. 在网格图上**点选/拖拽框选**一批点位 ID，并一键设为：
   - 障碍物
   - 待测区域
   - 清除定义

## 运行

```bash
streamlit run app.py
```

## 输入方式

### 方式 A：上传文件
- 支持：`.txt` / `.docx` / `.pdf`
- 上传后会自动解析成文本并展示在文本框，你可以继续手工修改。

### 方式 B：直接粘贴文本
建议包含如下关键词：
- `辐射超标地面点位ID`
- `辐射超标手套孔名称`

## 障碍物初始化

页面支持三种方式：
- 使用默认障碍物
- 手工输入障碍物 ID（空格 / 逗号 / 换行均可）
- 上传掩膜图（黑色 = 障碍物，白色 = 可达）

> 掩膜图建议与网格比例接近 `28:30`。

## 推荐操作流程

1. 上传或粘贴检测报告，确认自动识别结果。
2. 选择障碍物初始化方式。
3. 在图上拖拽框选点位。
4. 点击按钮设为障碍物、待测区域，或清除定义。
5. 如需重置，点击 `恢复默认障碍物`。

## 编号规则

- 每列 30 个点（行号 1~30）
- 下一列编号 +30
- 总点位数 = 30 × 28 = 840

例如：
- 第 1 列：1~30
- 第 2 列：31~60

## 注意

- PDF 解析依赖 `pypdf`，复杂扫描版 PDF（图片型）可能提取不到文字，建议先 OCR 后再导入。
- 掩膜图解析依赖 `pillow`。
