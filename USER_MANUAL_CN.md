## 一、引言

本说明书用于指导用户使用“辐射超标地面点位与手套孔研判平台”，完成检测报告接入、文本校核、网格区域修正、日期结果核查、多期汇总与趋势分析等业务流程。  
文档适用于业务操作人员、数据复核人员和分析研判人员。

---

## 二、用途

本软件主要用于以下场景：

1. 接入 `TXT / DOCX / PDF` 检测报告并自动识别污染点位与手套孔信息。  
2. 在 `30 × 28` 网格地图上进行污染、无污染、障碍与待测区域修正。  
3. 按日期查看单期识别结果与空间分布，支持结果核查。  
4. 对多期报告进行统一汇总，输出横向对照表。  
5. 开展趋势总览、热点画像、趋势预测与污染溯源分析，辅助排查决策。

## 四、使用说明

### 4.1 软件简介

系统采用单页分区式布局，按业务流程组织为五个功能段：  
一、报告接入与文本校核；二、区域定义与修正；三、日期结果；四、普查结果按日期汇总；五、趋势预测分析。

### 4.2 界面总览

页面顶部为流程化功能说明，主体区域按顺序分段展示。各段之间设置分隔线，便于区分不同业务功能。  
交互元素（按钮、标签页、导出控件）与说明性文本视觉分离，避免误判为可点击控件。

### 4.3 操作流程

#### 4.3.1 报告接入与文本校核

1. 上传一份或多份检测报告（支持 `txt/docx/pdf`）。  
2. 系统自动识别日期、污染点位 ID 与超标手套孔名称。  
3. 在“文本校核”区域修正报告文本，修正后识别结果实时同步。  
4. 若报告声明数量与识别数量不一致，系统提示复核预警。

#### 4.3.2 区域定义与修正

1. 在“区域基线设置”中选择障碍物来源：默认障碍、手工输入、掩膜导入。  
2. 在地图区域通过框选/套索多次累积选中点位。  
3. 在右侧“状态写入”中将选区写入污染、无污染、障碍、待测或清除定义。  
4. 必要时执行“取消选中 / 默认障碍 / 重置污染”等快速操作。

#### 4.3.3 日期结果核查

1. 查看当前激活日期的统计概览（污染点位、手套孔、障碍、待测）。  
2. 在结果明细区核对污染点位 ID、手套孔名称、行列映射。  
3. 对照地图空间位置，确认识别与修正结果一致性。

#### 4.3.4 多期汇总对照

1. 在“普查结果按日期汇总”查看多期结果横向对照。  
2. 核对最新值、峰值、累计新增/减少等汇总指标。  
3. 导出跨日期汇总 CSV 用于归档或二次分析。

#### 4.3.5 趋势预测分析

1. 打开趋势分析段，查看趋势总览、热点画像、趋势预测、污染溯源。  
2. 通过“分析参数与假设”调节预测窗口与权重参数。  
3. 结合回放控件按日期重演预测与溯源过程。  
4. 导出对应排行数据用于复核和报告引用。

## 五、各功能模块详解

### 5.1 报告接入与文本校核

1. 支持 `TXT / DOCX / PDF` 多格式自动抽取文本。  
2. 日期识别优先级：任务结束时间 > 正文日期 > 文件名日期。  
3. 自动识别关键字段：
   - 辐射超标地面点位 ID  
   - 辐射超标手套孔名称  
4. 支持人工修正文本文字，修正后污染点位集合自动重算。

### 5.2 区域定义与修正

1. 地图为 `30 × 28` 网格，共 `840` 点位。  
2. 编号规则：按列编号，每列 30 点；点位 ID 可映射到“行/列”。  
3. 支持多次拖拽累积选中，颜色先预览，再通过按钮确认写入。  
4. 状态写入逻辑：
   - 设为污染：加入污染集合，并移出障碍/待测  
   - 设为无污染：从污染集合移除  
   - 设为障碍/待测：写入对应集合并互斥处理  
   - 清除定义：从障碍/待测移除  

### 5.3 日期结果

1. 展示当前日期的核心统计卡片。  
2. 结果明细包含：
   - 当前污染点位 ID  
   - 超标手套孔名称  
   - 点位对应行列  
   - 当前报告状态  
3. 左侧地图与右侧明细协同展示，便于逐点核查。

### 5.4 普查结果按日期汇总

1. 按日期排序生成横向汇总表。  
2. 默认字段包括：
   - 日期  
   - 报告名称  
   - 污染地面点位数  
   - 超标手套孔数  
   - 较前一日变化  
   - 新增/减少点位数  
3. 支持导出 `cross_report_summary.csv`。

### 5.5 趋势预测分析

该模块包含四个标签页：

1. 趋势总览：分别展示污染点位总量、新增、减少、污染占比、重心迁移等。  
2. 热点画像：展示历史热点分布与热点排行。  
3. 趋势预测：基于历史频率、邻域支撑与连续性进行风险排序。  
4. 污染溯源：基于首次出现、持续性、扩散关系等生成疑似源点候选。

### 5.6 分析参数与假设

参数分为三组：趋势预测、风险预测、污染溯源。  
支持通过滑块调整窗口长度、各维度权重与阈值；参数变更后图表实时重算。

## 六、导出说明

系统支持导出以下 CSV 文件：

| 导出按钮 | 文件名 | 说明 |

## 七、注意事项

1. PDF 若为扫描图片型文档，可能无法直接抽取文本，建议先 OCR 后再导入。  
2. 掩膜图建议接近 `28:30` 比例；黑色像素表示障碍、白色像素表示可达。  
3. “状态写入”仅对当前选中点位生效，未选中点不会被修改。  
4. 趋势预测与溯源结果属于辅助研判结论，不替代现场复核。  
5. 建议在完成关键分析后及时导出 CSV，便于留档与复盘。  
6. 术语说明：
   - 污染点位：识别或修正后确认污染的网格点位  
   - 无污染点位：可达且未被标记污染的网格点位  
   - 障碍物点位：不参与可达判断的阻挡区域  
   - 待测区域点位：暂不纳入污染/无污染判断、等待复核的区域
